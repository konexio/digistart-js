Pour ouvrir la console JavaScript du navigateur

Faites un clic droit, puis Inspecter l'élément ou pressez F12 ou CTRL SHIFT J, et cliquez sur l'onglet "Console".

Rappels

Pensez à sauvegarder votre code dans VS Code en pressant CTRL S.

Pensez à rafraîchir votre page dans le navigateur (après avoir modifié et sauvegardé votre code) en pressant CTRL R ou F5.
